## 🏗️ Архитектура

### Backend (Go)
- REST API с Gorilla Mux
- PostgreSQL для хранения данных
- Prometheus метрики
- Структурированная архитектура (handlers, models, database, metrics)

### Frontend (Svelte)
- SPA с svelte-routing
- Современный минималистичный дизайн
- WCAG AAA доступность
- Responsive для всех устройств

### Мониторинг
- Prometheus для сбора метрик
- Grafana с готовыми дашбордами
- Метрики HTTP, бизнес-логики и БД

### Инфраструктура
- Docker Compose оркестрация
- Nginx для frontend
- Автоматизация через Makefile

## 🚀 Быстрый запуск

```bash
# Клонировать
git clone https://github.com/KapaSique/Department-boilerplate.git
cd Department-boilerplate

# Запустить
make up

# Инициализировать БД
docker exec -i finance-dept-db psql -U postgres -d finance_dept < backend/init.sql

# Открыть
open http://localhost:3000
```
