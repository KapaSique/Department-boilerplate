# 🚀 Быстрый старт

Автоматизированный скрипт для тестирования всего проекта.

## Использование

```bash
# Запустить Docker Desktop, затем:
./test.sh
```

## Что делает скрипт

1. ✅ Проверяет, что Docker запущен
2. 🧹 Очищает старые контейнеры и volumes
3. 🔨 Собирает все Docker образы
4. ▶️ Запускает все сервисы
5. ⏳ Ждет готовности сервисов
6. 💾 Инициализирует базу данных
7. ✅ Проверяет работоспособность всех сервисов
8. 🔍 Тестирует API endpoints
9. 📊 Выводит итоговый отчет

## Требования

- Docker Desktop должен быть запущен
- Порты 3000, 8080, 9090, 3001, 5432 должны быть свободны

## Результат

При успешном выполнении вы увидите:

```
🎉 Все тесты пройдены успешно!

Сервисы доступны по адресам:
  • Frontend:   http://localhost:3000
  • Backend:    http://localhost:8080
  • Prometheus: http://localhost:9090
  • Grafana:    http://localhost:3001 (admin/admin)
```

## Отладка

Если тесты провалились:

1. Проверьте логи: `docker-compose logs`
2. См. подробное руководство: [DOCKER_DEBUG.md](DOCKER_DEBUG.md)
3. Проверьте, что все порты свободны: `lsof -i :3000,8080,9090,3001,5432`

## Ручной запуск

Если предпочитаете запускать вручную:

```bash
# Сборка
docker-compose build

# Запуск
docker-compose up -d

# Инициализация БД
docker exec -i finance-dept-db psql -U postgres -d finance_dept < backend/init.sql

# Проверка
curl http://localhost:8080/api/v1/health
open http://localhost:3000
```
